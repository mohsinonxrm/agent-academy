    Welkom bij je volgende missie in de Copilot Studio Agent Academy. Je staat op het punt om de wereld van vooraf gebouwde agents te verkennen‚Äîintelligente, doelgerichte agents die door Microsoft zijn gemaakt om je implementatie te versnellen en de tijd tot waarde te verkorten.

    üß† Wat Zijn Vooraf Gebouwde Agents?¶

    Vooraf gebouwde agents zijn kant-en-klare AI-agents die door Microsoft zijn gemaakt en:

    • Inspelen op veelvoorkomende zakelijke behoeften (zoals reizen, HR, IT-ondersteuning)
    • Volledig functionerende onderwerpen, triggerzinnen, instructies en voorbeeldkennis bevatten
    • Kunnen worden bewerkt, uitgebreid en gekoppeld aan je eigen gegevens

    Deze agents zijn perfect om snel aan de slag te gaan of om te leren hoe agents zijn opgebouwd.

    We blijven bij het eerdere voorbeeld, waarin we een oplossing maken in de speciale Copilot Studio-omgeving om onze IT-helpdeskagent te bouwen.

    Laten we beginnen!

    5.1 Start Copilot Studio¶

    1. Ga naar https://copilotstudio.microsoft.com

    2. Log in met je Microsoft 365 werk- of schoolaccount

    Warning

    Je moet je bevinden in een tenant waar Copilot Studio is ingeschakeld. Als je Copilot Studio niet ziet, ga dan terug naar Missie 00 om je setup te voltooien.

    5.2 Kies de Veilige Reizen Agent-sjabloon¶

    1. Klik op de startpagina van Copilot Studio op + Maken

    2. Scroll naar beneden naar de sectie Begin met een agent-sjabloon

    3. Zoek en selecteer Veilige Reizen

    4. Merk op dat de sjabloon vooraf is geladen met een beschrijving, instructies en kennis.

    5. Klik op Maken

    Dit maakt een nieuwe agent in je omgeving op basis van de configuratie van Veilige Reizen.

    5.3 Pas de Agent aan¶

    Nu de agent is gemaakt, gaan we deze aanpassen aan jouw organisatie:

    1. Selecteer Generatieve AI ingeschakeld om de generatieve AI-functie in te schakelen, zodat deze de instructies in de sjabloon kan gebruiken.

    2. We gaan de agent nu voorzien van een extra kennisbron, zodat deze vragen over reizen in Europa kan beantwoorden. Scroll hiervoor naar beneden naar de sectie kennis en selecteer Kennis toevoegen

    3. Selecteer Openbare websites

    4. Plak in het tekstveld https://european-union.europa.eu/ en selecteer Toevoegen

    5. Selecteer Toevoegen aan agent

    5.4 Test en Publiceer¶

    1. Klik rechtsboven op Testen om het testvenster te openen

    2. Probeer zinnen zoals:

      • ‚ÄúHeb ik een visum nodig om van de VS naar Amsterdam te reizen?‚Äù
      • ‚ÄúHoe lang duurt het om een Amerikaans paspoort te krijgen?‚Äù
      • ‚ÄúWaar is de dichtstbijzijnde Amerikaanse ambassade in Valencia, Spanje?‚Äù

    3. Controleer of de agent reageert met nauwkeurige en nuttige informatie en observeer de Activiteitenkaart om te zien waar de informatie vandaan komt.

    4. Klik op Publiceren wanneer je klaar bent

    5. Selecteer opnieuw Publiceren in het dialoogvenster

    6. Voeg de agent optioneel toe aan Microsoft Teams met behulp van de ingebouwde Kanalen-functie.
